Task Manager CLI
A simple and efficient command-line interface (CLI) application to manage your daily tasks. This project implements a CRUD system (Create, Read, Update, Delete) using Python.
Features

    Create tasks: Add new activities to your list.
    List tasks: View all your pending and completed tasks.
    Search: Quickly locate specific tasks.
    Update: Modify the content or status of your tasks.
    Delete: Remove tasks you no longer need.
    Data persistence: Tasks are automatically saved upon exiting.

Project Structure
The project is organized into modules to facilitate maintenance:

    main.py: Application entry point and main menu logic.
    functions.py: Contains the logic for task manipulation (create, edit, etc.).
    services.py: Manages data loading and saving (persistence).
